# Анализатор метаданных файлов

Код для анализа, проверки и очистки метаданных различных типов файлов.

## Функционал

- **Извлечение метаданных** из файлов разных форматов
- **Обнаружение опасной информации** (GPS, личные данные)
- **Очистка метаданных** перед публикацией файлов
- **Пакетная обработка*** файлов в папках
- **Валидация и обработка ошибок**

## Поддерживаемые форматы

| Формат       | Что анализируется                               |
| ------------ | ----------------------------------------------- |
| JPG/JPEG/PNG | EXIF данные, GPS координаты, параметры камеры   |
| PDF          | Автор, название, дата создания, производитель   |
| DOCX         | Свойства документа, автор, даты, ключевые слова |

## Установка

### 1. Установка зависимостей
pip install -r requirements.txt

### 2. Проверка установки
python examen.py help

## Использование

### 1.Анализ одного файла
python examen.py analyze фото.jpg

### 2.Сканирование папки
python examen.py scan С:/Documents

### 3.Очистка метаданных изображения
python examen.py clean изображение.jpg

### 4.Показать справку
python examen.py help
